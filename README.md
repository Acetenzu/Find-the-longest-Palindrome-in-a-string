# Find-the-longest-Palindrome-in-a-string
This can helps you to find the longest palindrome in a string
要在string中找出最長迴文(正著讀和反著讀一樣，且長度必須 ≥ 2)
Input
輸入一行字串（長度 ≤ 50，只包含小寫英文字母）
Output
⚫ 輸出一行字串，為最長的迴文子字串。
⚫ 不存在有效迴文，則輸出 No Palindrome。

Sample Input 1
marioracecar
Sample Output 1
1racecar

Sample Input 2
abcddcbaxyz
Sample Output 2
abcddcba

Sample Input 3
justdoit
Sample Output 3
No Palindrome

Sample Input 4
a
Sample Output 4
No Palindrome
